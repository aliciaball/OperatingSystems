Memory Management

Memory Management is the function used to keep track of moving processes between the main memory and the disk memory.
Memory Management also keeps track of the memory location processes and how much of it is free. 
It also delegates what processes will get memory and at what time while keeping it's status up to date. 

Processor Management

Processor Management is in charge of keeping track of prcoess statuses. 
It also assigns processes within the CPU.
It will also unassign unused processes.

File Management

File Management is the function that keeps track of many things like information, status and location.
It also decides who or what gets the resources.
File management ultimately decides who is assigned and unassigned the resources.
